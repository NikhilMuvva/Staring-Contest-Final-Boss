# Code of Conduct

## Our Pledge

We want this project to be welcoming, respectful, and useful for collaborators, students, makers, and exhibit visitors.

## Expected Behavior

- Be respectful and constructive.
- Give clear feedback without personal attacks.
- Assume good intent.
- Keep discussions focused on improving the project.

## Unacceptable Behavior

- Harassment or discriminatory language.
- Personal insults.
- Trolling or intentionally disruptive behavior.
- Sharing private information without permission.

## Reporting

If there is a problem, please contact the project maintainer or open a private communication channel if one is available.

## Enforcement

Maintainers may remove comments, close issues, or block contributors who do not follow these expectations.
